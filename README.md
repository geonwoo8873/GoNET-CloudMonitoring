# Custom-GitHub-Copilot-Rule

## Ko

> [!NOTE]
> 이 템플릿은 현재 사용중인 IDE내에서 코파일럿의 무분별한 토큰값을 학습 방지와 로컬 통해서 코드 제안 및 채팅에 대한 데이터 저장을 방지하여 다음과 같이 커스터마이즈 룰을 적용하는 계획입니다.

### 적용 범위 [`2026. 04. 21`]

* Environment values
* JSON values
* API Keys
* Passwords
* Session Logs
* Tokens

> [!WARNING]
> 향후 적용 방식과 룰은 변경될 수 있으며 사용자에 따라 불편함을 야기할 수 있음에 주의 부탁드립니다.
